# Playbook Comercial de Diamond Inmobiliaria

Sistema completo de generación de clientes vía Meta Ads para venta, alquiler tradicional y rentas cortas en Medellín. Construido como los 8 documentos especializados de un playbook comercial real (no un único informe monolítico), para que cada uno se pueda actualizar, imprimir o entregar por separado a quien lo necesite (agencia de pauta, asesor nuevo, diseñador).

## Principio rector

Cada documento distingue explícitamente:
- **Dato verificado** (con fuente) — cifras de Medellín, funcionamiento actual de Meta Ads.
- **Principio extraído** de inmobiliarias de lujo (Sotheby's, Compass, SERHANT, etc.) — nunca una copia de sus anuncios, sino el porqué detrás de lo que funciona, adaptado a Medellín.
- **Recomendación del equipo** — la síntesis aplicada a Diamond.

## Cómo usar este playbook

- Si eres asesor nuevo: empieza por el Doc 6 (Manual de Ventas) y el Doc 2 (para entender a quién le hablas).
- Si vas a lanzar pauta: Doc 3 (Meta Ads) + Doc 4 (Creativos) + Doc 8 (presupuesto según lo que tengas para invertir).
- Si administras el negocio: Doc 7 (Dashboard) es tu vista semanal.
